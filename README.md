## Youchu 소개
유추는 사용자의 유튜브 구독목록을 기반으로 협업 필터링(Collaborative Filtering)을 통해 맞춤형 채널들을 추천해줍니다.
1. 맟춤으로 추천되는 채널을 살펴보세요
2. 랜덤으로 추천되는 채널들을 탐색할 수 있습니다
3. 카테고리 별 랭킹(구독자 순)을 확인해 보세요.
<br><br>

### 개발 기간
약 2달 (4월 ~ 5월 말)

### 기술 스택
- UIKit
- Alamofire (RestApi 통신)

### 클라이언트 주요 기능
- 채널 상세 페이지 Sticky Header 구현
- 구글, 애플 로그인 구현
- 토큰을 활용한 Rest API 통신, Interceptor 사용

### Tools Used
- Xcode 12.4

### Application Version
- Over IOS 13
